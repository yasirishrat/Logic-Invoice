Logic Invoice Accounting

Accounts


In this page, you will find all the accounts that will be used in your accounting. By default, there are already accounts created for you. You may wish to add/edit/delete accounts to suit your needs.

Adding an Account


Account ID This is used in the chart of accounts to help with the arrangement of the accounts
Name This is the name of the account
Description Description of the account (optional)
Type Choose a type for the account. Do note that choosing the wrong time will lead to inaccuracies in the reporting system
Parent Account Choose whether this account has a parent account
Status Choosing whether this account is enabled or disabled

Deleting Accounts

You can select the accounts that you want to delete, and click on the red button with the trash bin.

Editing an Account

You can edit the account by clicking on the button with the pencil on the right hand side of the account.

Currencies


The Currencies page contains a list of currencies that you want to do your accounting in. When you click on the Refresh button, Logic Invoice will automatically update the values of the currencies using data from Yahoo! Finance.

Adding Currency


Title Name of the currency you are adding
Code Currency code based on the ISO 4217 Currency Codes. Do note that if this is not correct, Logic Invoice will be unable to extract the correct currency values.
Symbol Left Symbol that you want to display on the left hand side of the currency amount
Symbol Right Symbol that you want to display on the right hand side of the currency amount
Decimal Place Number of decimal places to show the currency in
Value Value of this currency against your default currency which has a value of 1
Status Choosing whether this currency is enabled or disabled

Deleting Currencies

You can select the currencies that you want to delete, and click on the red button with the trash bin.

Editing a Currency

You can edit the currency by clicking on the button with the pencil on the right hand side of the currency.

Journal Entries


This is where you will be able to view all the journal entries previously added.

Adding Journal Entries


Fill in all the required fields on the top of the page. You may add the accounts involved in this journal entry by clicking on the green plus button.

Deleting Journal Entries

You can select the journal entries that you want to delete, and click on the red button with the trash bin.

Editing a Journal Entry

You can edit the journal entry by clicking on the button with the pencil on the right hand side of the journal entry.

Taxes - Tax Classes


The Tax Classes page lists all the tax classes that are available to be applied.

Adding Tax Classes


Fill in all the required fields, and add a tax rate to the tax class by clicking on the green plus button.

Deleting Tax Classes

You can select the tax classes that you want to delete, and click on the red button with the trash bin.

Editing a Tax Class

You can edit the tax class by clicking on the button with the pencil on the right hand side of the tax class.

Taxes - Tax Rates


The Tax Rates page lists all the tax rates that are available to be added to Tax Classes.

Adding a Tax Rate


Name Name of the tax rate
Rate Numerical value
Type Choose whether this tax is applied at a fixed rate, or a fixed percentage

Deleting Tax Rates

You can select the tax rates that you want to delete, and click on the red button with the trash bin.

Editing a Tax Rate

You can edit the tax rate by clicking on the button with the pencil on the right hand side of the tax rate.
